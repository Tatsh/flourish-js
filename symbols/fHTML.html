<static> {boolean} fHTML.containsBlockLevelHTML(content, strict)
Checks a string of HTML for block level elements. Elements that are by default not counted as block-level:
  • <link>
  • <nav>
  • <optgroup>
  • <option>
  • <summary>
  • <tbody>
  • <td>

To check for these elements, set the strict argument to true.

Note that in IE versions prior to 9, because this uses innerHTML, HTML 5 elements and other unknown elements do not work unless a solution like html5shiv is employed (Modernizr comes with this by default).

html5shiv

Parameters:
{string} content
The HTML content to check.
{boolean} strict Optional, Default: false
Do not allow exceptions.
Returns:
{boolean} If the content has a block level tag.
